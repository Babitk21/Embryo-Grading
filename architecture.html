System Architecture

End-to-end AI pipeline for blastocyst quality assessment.

Overall Pipeline

The system follows a structured pipeline starting from embryo image acquisition and ending with interpretable grading outputs. Each stage is designed to ensure consistency and reproducibility.

Pipeline Flow:
Image Input → Preprocessing → Deep Learning Model → Multi-Output Predictions → Inference Interpretation → User Interface

Image Preprocessing

Input embryo images are resized, normalized, and standardized to ensure uniform input distribution for the model. This step reduces noise and improves learning stability.

Model Design

A convolutional neural network (CNN) is used to extract spatial and morphological features from embryo images. The network produces independent predictions for:

  • Inner Cell Mass (ICM)
  • Trophectoderm (TE)
  • Expansion (EXP)

Inference Layer

Raw model outputs are post-processed into interpretable grades. This allows the system to present predictions in a form that aligns with clinical grading standards.
